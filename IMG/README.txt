🎨 LƯU Ý KHI CHỌN ẢNH:
=====================

Avatar (300x300):
✓ Ảnh rõ nét, chất lượng cao
✓ Khuôn mặt/logo ở giữa
✓ Nền đơn giản hoặc blur
✗ Tránh ảnh quá tối
✗ Tránh chi tiết quá nhiều

Background (1920x1080):
✓ Độ phân giải cao
✓ Không quá rối mắt
✓ Tone màu tối (dễ đọc text)
✗ Tránh ảnh quá sáng
✗ Tránh pattern lặp đi lặp lại

⚙️ CÁCH SỬ DỤNG:
===============

1. Đặt file ảnh vào thư mục IMG/
2. Đặt tên file theo qui ước trên
3. Mở gaming-media-contact.html
4. Trong Settings Panel:
   - Upload ảnh trực tiếp, HOẶC
   - Gõ: IMG/avatar1.jpg (example)
5. Save và Preview

💡 TỐI ƯU HÓA ẢNH:
=================

Trước khi upload lên website thật:
- Nén ảnh để giảm dung lượng (TinyPNG.com)
- Avatar: < 200KB
- Background: < 500KB
- Convert sang WebP nếu có thể (hiệu suất tốt hơn)

📏 KÍCH THƯỚC FILE:
==================

Để web load nhanh:
- Avatar: 50-200KB
- Background: 200-500KB
- Tổng tất cả ảnh: < 1MB

🔗 PATH TRONG CODE:
==================

Trong settings, dùng:
✓ IMG/avatar1.jpg
✓ IMG/background.png
✗ /IMG/avatar1.jpg (sai)
✗ img/avatar1.jpg (sai, phân biệt hoa thường)
